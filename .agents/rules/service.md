# Cloud Memo — 서비스 컨텍스트 문서

> **슬로건:** 생각을 클라우드에 담다

## 서비스 개요

Cloud Memo는 개인 및 팀을 위한 **클라우드 기반 메모·노트 SaaS 서비스**입니다. 모든 기기에서 실시간으로 동기화되는 스마트 메모 환경을 제공하며, AI 기반 자동 분류, 강력한 검색, 풍부한 텍스트 편집, 파일 첨부 기능을 갖추고 있습니다. 한국 시장을 주요 타깃으로 하며, 한국어 UI와 카카오·네이버 소셜 로그인을 지원합니다.

---

## 핵심 가치 제안

| 구분 | 설명 |
|---|---|
| **실시간 동기화** | 스마트폰, 태블릿, PC 등 모든 기기에서 변경사항이 즉시 반영 |
| **스마트한 정리** | AI가 메모 내용을 분석하여 카테고리 자동 분류 및 태그 추천 |
| **안전한 보안** | AES-256 군사급 암호화, 종단간(E2E) 암호화로 개인 정보 완벽 보호 |
| **오프라인 지원** | 네트워크 미연결 상태에서도 메모 작성 및 편집 가능 |

## 페이지 구성 (디자인 목업 기준)

### 1. 랜딩 페이지 (`landing.html`)

서비스의 첫 인상을 담당하는 **마케팅 랜딩 페이지**입니다.

**주요 섹션:**
- **네비게이션 바:** 로고, 기능·가격·소개 링크, "시작하기" CTA 버튼. Sticky 헤더로 스크롤 시에도 상단 고정.
- **히어로 섹션:** `New Evolution of Note` 뱃지, 메인 헤드라인 ("클라우드에 생각을 담다, Cloud Memo"), 서브 문구, "무료로 시작하기" + "데모 보기" CTA, 사용자 아바타와 "10,000명 이상 사용 중" 소셜 프루프, 앱 대시보드 미리보기 일러스트레이션.
- **기능 소개 섹션 (`#features`):** 실시간 동기화 / 스마트한 정리 / 안전한 보안 — 아이콘 카드 3개 격자 배치. 호버 시 카드 상승 애니메이션.
- **CTA 배너 섹션:** Primary 컬러 그라디언트 배경. "기록의 시작이 당신의 성장이 됩니다" Copy, "지금 무료 시작" + "요금제 보기" 버튼.
- **푸터:** Cloud Memo 소개, 제품·리소스·법률 링크 컬럼, 저작권.

**디자인 특징:**
- 다크 모드 지원 (`darkMode: "class"`)
- 배경색 `#fcfcf2` (크림 화이트) / 다크 배경 `#1a1c1a`
- 브랜드 컬러 Primary `#4e6e52` (포레스트 그린 계열)
- 폰트: Inter + Noto Sans KR

### 2. 인증 페이지 (`auth.html`)

로그인 및 회원가입을 처리하는 **인증 페이지**입니다.

**레이아웃:** 좌우 2-컬럼 분할 레이아웃 (데스크톱 기준)
- **좌측 패널:** 브랜드 컬러 메시 그라디언트 배경, Cloud Memo 로고 및 슬로건, "1만 명 이상 신뢰" 소셜 프루프, 사용자 아바타 3개.
- **우측 패널 (폼 영역):**
  - 이메일 + 비밀번호 입력 필드 (비밀번호 표시/숨김 토글 버튼)
  - "비밀번호를 잊으셨나요?" 링크
  - "30일 동안 로그인 유지" 체크박스
  - 로그인 버튼 (Primary 컬러)
  - 소셜 로그인: **카카오** / **네이버** (한국 특화)
  - 하단 회원가입 유도 링크

### 3. 대시보드 페이지 (`dashboard.html`)

로그인 후 사용자가 처음 도달하는 **메인 대시보드**입니다.

**레이아웃:** 좌측 사이드바 + 우측 메인 콘텐츠

**사이드바 항목:**
- 로고 + 플랜 표시 (Professional Plan)
- 네비게이션: 대시보드 / 모든 메모 / 공유됨 / 저장공간 / 설정
- 하단: 저장공간 사용량 진행 바(예: 10GB 중 2.5GB), 결제 관리 버튼

**메인 콘텐츠 영역:**
- **헤더:** 메모 검색창, 알림 버튼 (빨간 점 알림 뱃지), 사용자 이름 + 프로필 아바타
- **환영 메시지:** 사용자 이름 기반 인사, "오늘의 생산성 요약"
- **현황 카드 그리드:**
  - 구독 카드 (플랜명, 갱신일, "결제 관리" + "내역 보기" 버튼)
  - 이달의 사용량 카드 (클라우드 저장공간 / API 호출 진행 바)
- **최근 활동 목록:** 최근 수정된 메모 4건 (종류별 아이콘 색상 구분: 파랑/주황/초록/보라), 공유 협업자 아바타 표시
- **하단 통계:** 전체 메모 수 / 공유된 메모 수 / 생산성(%) / 주간 단어 수

### 4. 메모 편집 페이지 (`note.html`)

실제 메모를 열람하고 작성하는 **핵심 편집 화면**입니다.

**3-패인(Pane) 레이아웃:**

| 패인 | 역할 | 주요 요소 |
|---|---|---|
| **패인 1 — 사이드바 폴더** | 폴더 및 분류 탐색 | 라이브러리(All Cloud / 메모 / 공유됨), 사용자 폴더(업무 프로젝트, 개인적인 아이디어, 여행 기록), 보관함, 최근 삭제된 항목 |
| **패인 2 — 메모 목록** | 선택한 폴더의 메모 리스트 | 메모 제목 + 미리보기 텍스트 + 날짜/시간, "새 메모" 버튼, 선택 메모 좌측 강조 테두리 |
| **패인 3 — 에디터** | 메모 내용 편집 | 리치 텍스트 툴바(Bold/Italic/Underline/목록/정렬/이미지/링크/코드), 제목(contenteditable h1), 본문 편집 영역, 이미지 삽입 지원 |

**에디터 툴바 기능:**
- 텍스트 서식: 굵게, 기울임체, 밑줄
- 목록: 불릿 목록, 번호 목록, 체크리스트
- 정렬: 좌측, 가운데
- 미디어: 이미지 삽입, 링크 삽입, 코드 블록
- 동기화 상태 표시: "방금 동기화됨" (실시간)

**지원 콘텐츠 타입:**
- 일반 텍스트
- 제목 구조 (h3 등)
- 체크리스트 아이템 (완료/미완료 상태)
- 인용문 블록
- 이미지

**하단 상태 바:** 폴더 수 / 메모 수 / 시스템 상태 ("모든 시스템 정상 작동 중")

### 5. 요금제·결제 페이지 (`payment.html`)

구독 플랜을 선택하는 **가격 페이지**입니다.

**플랜 구성:**

| 플랜명 | 가격 | 주요 기능 | CTA |
|---|---|---|---|
| **베이직 (Basic)** | $0 / 월 (무료) | 5GB 저장공간, 2기기 동기화, 기본 검색 | 시작하기 |
| **프로 (Pro)** | $9.99 / 월 | 무제한 저장공간, 모든 기기 동기화, 우선 지원, 이미지·PDF OCR, 고급 서식 | 플랜 선택하기 ⭐ 가장 인기 |
| **팀 (Team)** | $29.99 / 월 | Pro 모두 포함, 관리자 대시보드, 팀 협업, SSO 로그인, 맞춤형 온보딩 | 영업 문의 |

**FAQ 섹션:**
- 구독 취소 정책 (언제든 취소 가능, 현재 기간 종료까지 이용)
- 결제 수단 (Visa, Mastercard, Amex, PayPal, Apple Pay; 연간 결제 시 청구서 발행)
- 할인 정책 (학생·비영리 단체 50% 할인)

**하단 CTA 배너:** "고객 지원 문의" 버튼

## 디자인 시스템

### 컬러 팔레트

| 토큰 | 값 | 용도 |
|---|---|---|
| `primary` | `#4F6F52` | 브랜드 컬러, 버튼, 링크, 강조 |
| `background-light` | `#fcfcf2` | 라이트 모드 배경 (크림 화이트) |
| `background-dark` | `#1a1c1a` | 다크 모드 배경 |

### 타이포그래피
- **English:** Inter (300~900 웨이트)
- **Korean:** Noto Sans KR (300~900 웨이트)
- **아이콘:** Google Material Symbols Outlined

### 공통 UI 패턴
- 다크 모드: `class` 기반 (`dark:` 접두사)
- 반응형: Tailwind CSS 브레이크포인트 (`sm:`, `md:`, `lg:`)
- 버튼 호버: `scale-105` / `active:scale-95` 마이크로 인터랙션
- 글래스모피즘: `backdrop-blur-md` + 투명도 배경
- 카드: `rounded-xl`, `border border-primary/10`, `shadow-sm`

## 기술 스택 (디자인 기준)

- **CSS 프레임워크:** Tailwind CSS (CDN, forms + container-queries 플러그인)
- **아이콘:** Google Material Symbols Outlined
- **폰트:** Google Fonts (Inter, Noto Sans KR)
- **다크 모드:** CSS 클래스 기반 (`class` strategy)
- **UI 언어:** 한국어 (ko), 일부 영문 혼용

---

## 주요 기능 요약

1. **메모 작성:** 리치 텍스트 에디터 (Bold/Italic/목록/체크리스트/이미지/코드 지원)
2. **폴더 관리:** 사용자 정의 폴더 생성 및 계층적 메모 분류
3. **실시간 동기화:** 모든 기기 간 즉각적 동기화, 오프라인 우선 아키텍처
4. **공유 협업:** 메모 또는 노트북 단위 공유, 협업자 실시간 공동 편집
5. **AI 태그 추천:** 내용 기반 자동 카테고리 분류 및 태그 추천
6. **검색:** 전체 메모 내 전문 검색 (Pro: OCR 기반 이미지·PDF 내 텍스트 검색)
7. **저장공간 관리:** 플랜별 저장 한도 및 사용량 모니터링
8. **구독 관리:** 플랜 변경, 결제 내역 조회, 구독 취소
9. **소셜 로그인:** 이메일/비밀번호, 카카오, 네이버
10. **다크 모드:** 시스템 또는 수동 전환 지원
